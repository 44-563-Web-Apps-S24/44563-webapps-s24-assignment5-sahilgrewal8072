As you resize the browser window, you'll observe that the row featuring the drink names (configured with col-md classes) is typically the first to break. This behavior is due to the fact that the row containing letters (set with col-sm-2 classes) is designed to adjust at smaller screen sizes (sm), while the drink names row adapts at medium screen sizes (md). Consequently, the drink names row is more likely to collapse before the letters row as the screen size decreases. The phenomena is carried by bootstrap classes making it the responsive design.
